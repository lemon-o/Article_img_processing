---
一个将文章所需图片素材统一处理的脚本
---
能够将图片统一尺寸并添加投影效果<br>
<br>
使用说明：<br>
1、python官网下载python3<br>
2、打开cmd使用命令：pip install pillow<br>
3、将脚本放到图片所在目录，双击运行即可

支持的格式：'.png', '.jpg', '.jpeg'
